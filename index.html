Privacy Policy

This Privacy Policy explains how information is handled when users access and use the Sheikh Mahad Gurey mobile application.

Effective Date: 21 July 2026
Last Updated: 21 July 2026

The Sheikh Mahad Gurey application respects the privacy of its users. This Privacy Policy explains how information is handled when users access and use the Sheikh Mahad Gurey mobile application.

1. About the Application

Sheikh Mahad Gurey is an Islamic educational mobile application featuring Quranic Tafsir lessons and audio recordings by Sheikh Mahad Gurey, a Somali Islamic scholar.

The application allows users to browse, read, download available content, and listen to Islamic educational materials.

2. Personal Information

The application does not require users to create an account, register, or sign in.

The application does not directly collect personal information such as:

  • Names
  • Email addresses
  • Phone numbers
  • Physical addresses
  • Contact lists
  • Photos or personal files
  • Precise location
  • Financial or payment information

We do not sell, rent, trade, or use personal information for advertising or marketing purposes.

3. Internet Access and Content Delivery

The application may use an internet connection to retrieve Quranic Tafsir audio recordings, images, and related educational content from cloud-hosted storage.

Application content may be delivered through Cloudflare R2 or similar content-hosting infrastructure.

Cloud hosting providers and internet service providers may automatically process limited technical information when delivering content, such as:

  • IP addresses
  • Network request information
  • Device connection information
  • Request dates and times

This technical information may be processed only as necessary to securely host and deliver application content.

The application does not use this information for advertising, behavioural profiling, or user tracking.

4. Information Stored on the Device

The application may store limited information locally on the user’s device, including:

  • Downloaded audio lessons
  • Cached audio and images
  • Bookmarked lessons
  • Listening progress
  • Playback preferences
  • Application settings

This information is stored locally to support the application’s functionality and improve the user experience.

Locally stored information is not transmitted to the developer’s servers.

Users may remove locally stored information by:

  • Deleting downloaded content
  • Clearing the application’s storage or cache
  • Resetting the application
  • Uninstalling the application

5. Advertising

The application does not display third-party advertisements.

The application does not use personal information for advertising, targeted advertising, or promotional profiling.

6. Analytics and Tracking

The current version of the application does not use:

  • Advertising trackers
  • Behavioural tracking technologies
  • User profiling systems
  • Third-party advertising analytics

The application does not track users across other applications or websites.

If analytics or tracking technologies are introduced in a future version, this Privacy Policy will be updated before those services are used.

7. Data Sharing

The application does not sell, rent, or share users’ personal information with third parties.

Technical service providers may process limited network information only when necessary to:

  • Host application content
  • Deliver audio and images
  • Maintain service security
  • Prevent abuse
  • Ensure reliable content delivery

No user information is shared for advertising or marketing purposes.

8. Data Security

Reasonable technical measures are used to protect the application and its content.

Where supported, application content is delivered through secure and encrypted network connections.

However, no method of internet transmission or electronic storage can be guaranteed to be completely secure.

9. Children’s Privacy

The application does not knowingly collect personal information from any users, including children.

Users can access the application’s educational content without creating an account or submitting personal information.

Parents or guardians who believe that personal information has been submitted through the application may contact the developer using the details provided below.

10. Data Retention and Deletion

The application does not maintain a central user account database.

Locally stored application information remains on the user’s device until the user:

  • Deletes downloaded content
  • Clears the application data
  • Resets the application
  • Uninstalls the application

Information temporarily processed by content-hosting providers may be retained according to those providers’ security and operational policies.

11. Changes to This Privacy Policy

This Privacy Policy may be updated when:

  • Application features change
  • New services are introduced
  • Data-handling practices change
  • Legal or platform requirements change

Any revised version will be published on this page with an updated effective date.
